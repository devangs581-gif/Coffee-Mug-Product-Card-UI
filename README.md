# ☕ Coffee Mug Product Card UI

A clean and modern **Product Card UI** showcasing a coffee mug with pricing, description, and add-to-cart functionality. Built using HTML & CSS with a minimal and elegant design.

---

## 🌐 Live Demo

https://coffee-mug-product-card-ui-ueqs.vercel.app/

---


## ✨ Features

- 🛍️ Product card layout with image and details
- 💰 Pricing section with discount display
- 📝 Product description
- 🛒 Add to Cart button
- 🎨 Clean and minimal UI design
- 📱 Responsive layout

---

## 🛠️ Tech Stack

- HTML5
- CSS3

---



## 🎨 UI Highlights

- Split layout (image + content)
- Soft shadow and rounded corners
- Discounted price styling
- Clean typography
- Modern button design
